# 🚀 SHORT-V-DL (Short Video Downloader)

轻量级、智能化的自媒体短视频批量提取与下载工具。

目前首发核心功能：**抖音 (Douyin) 无水印视频批量提取与全自动保存**

---

## ⚡ 极其简单的部署说明 (One-Click Deploy)

本项目采用现代化的前后端解耦架构（React 前端界面 + Python/FastAPI 后端引擎）。为了让您免去繁琐的配置，我们为您提供了**一键安装和启动的傻瓜式脚本**。

### 📋 环境要求
- 你的操作系统已安装 **Python** (建议 3.10+) 
- 你的操作系统已安装 **Node.js** (建议 16+)

### 🛠️ 快速启动

**1. 准备一键启动套件**

- **Mac / Linux 用户**: 在终端（Terminal）赋予脚本运行权限：
  ```bash
  chmod +x start.sh stop.sh
  ```
- **Windows 用户**: 您无需赋予权限，可直接双击运行 `.bat` 批处理文件。

**2. 启动服务**

- **Mac / Linux**: 执行 `./start.sh`
- **Windows**: 直接双击运行包内的 `start.bat`

> **首次运行提醒**：脚本初次启动时需要几分钟时间。它会自动穿越到前端和后端目录，执行所有底层依赖的下载安装 (`npm install` & `uv sync/pip`)，并自动为您本地安装必须的无头浏览器内核 (`playwright install`)。配置完毕后，它会自动并排拉起各项服务集群。

**3. 开始使用**

服务成功启动后，直接在浏览器中访问您的前端可视化控制台：
👉 **http://localhost:8080** 

**4. 结束使用**

当您使用完毕，想彻底切断后台的爬虫引擎并释放被占用的端口资源时，只需一键化清理：
```bash
./stop.sh
```

## 📁 核心目录结构

本项目严格遵循高内聚低耦合的架构规范进行了深度瘦身重构。

- `/start.sh` & `/stop.sh`: 项目的命脉，跨语言环境一键启停脚本。
- `/RULES.md`: 本项目遵循的 AI 全局协作与开发编码规范。
- `/src/modules/web-ui/`: 基于 `React` + `Vite` 构建的现代化前端纯视觉交互层。
- `/src/modules/media-crawler/`: 基于 `Python` + `FastAPI` 构建的核心下载调度 API，内部承载了强悍的无头浏览器实例与各平台复杂的 JS 签名解析算法。
- `/docs/`: 提供给二次开发者的架构模块文档。

## 💡 使用小贴士 (Tips)

关于防封与 Cookie 保护：
当您在新版 Web 界面中点击**「拉起扫码登录」**时，后台会在您的本地真实弹出一个纯净的无痕浏览器供您用手机扫码。
扫码成功获取身份后，您可以尽情地批量下载视频。使用完毕后，只需点击**「关闭后台浏览器」**按钮即可悄声无息地释放内存。系统会在本地妥善为您加密缓存这一次的成功会话（Cookie），方便几天内免扫码复用！
