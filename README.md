# 🚀 SHORT-V-DL (Short Video Downloader)

轻量级、智能化的自媒体短视频批量提取与下载工具。

目前首发核心功能：**抖音 (Douyin) 无水印视频批量提取与全自动保存**

---

## ⚡ 极其简单的部署说明 (One-Click Deploy)

本项目采用现代化的前后端解耦架构（React 前端界面 + Python/FastAPI 后端引擎）。为了让您免去繁琐的配置，我们为您提供了**一键安装和启动的傻瓜式脚本**。

### 📋 环境要求
- 你的操作系统已安装 **Python** (建议 3.10+) 
- 你的操作系统已安装 **Node.js** (建议 16+)

### 🛠️ 快速启动

**1. 初次运行：一键安装环境**

对于刚下载本项目的新用户，请一定**首先双击运行或执行安装程序**！
- **Mac / Linux 用户**: 在终端赋予权限后执行：`chmod +x install.sh start.sh stop.sh`，然后运行 `./install.sh` 
- **Windows 用户**: 直接双击运行包内的 `install.bat` 批处理文件

> **安装提示**：环境安装脚本需要为您静默下载 `node_modules` 以及 `Playwright` 的底座 200MB 专用反爬浏览器内核。根据网络速度，安装**需要耗时两到三分钟**。看到终端提示“所有环境均已彻底安装成功”则表示大功告成！

**2. 日常启动服务**

只要您完成过一次环境安装，**以后任何时候使用本工具**，都享受到“零延迟”的闪电启动：
- **Mac / Linux**: 执行 `./start.sh`
- **Windows**: 直接双击运行 `start.bat`

**3. 开始使用 (Usage Guide)**

服务成功启动后，请在浏览器中访问可视化控制台：
👉 **http://localhost:8080** 

- **第一步：环境鉴权 (扫码登录)**
  1. 点击控制台界面上的 **“拉起扫码登录 (自动获取 Cookie)”** 按钮。
  2. 此时系统会在您的电脑后台安全环境中弹出一个真实的无痕浏览器，并打开抖音网页版。
  3. 请打开您的手机抖音，对准弹出的浏览器屏幕**扫码登录**。手机确认登录后，页面会自动提取通过安全令牌校验的 Cookie，并将您的身份 Cookie 安全折叠保管在本地。
  4. 提取成功后，点击界面的 **“关闭后台浏览器”** 释放系统内存。在凭证过期前的相当长一段时间内，您都无需重新扫码！

- **第二步：填入目标**
  将您在手机上点击分享获取到的**口令杂乱文案**，或是直接获取的**多条长短链接**，一次性粘贴进顶部的**“分享文案/链接”**多行输入框中。您可以贴十几段毫不相干的文案混排，AI 都会智能帮您过滤分离开来。

- **第三步：智能单页解析**
  点击 **“全部解析 (Parse All)”** 按钮。程序会毫秒级剥离所有干扰文字，还原出真实的视频链接，并在下方为您呈现排版好的目标资源序列卡片。

- **第四步：一键无损下载**
  确认解析无误后，点击总控台的 **“下载全部”**，或针对单张卡片点击 **“下载该项”**。绝佳画质的无水印视频文件会直接传输并保存至您的电脑本地。若遇网络异常导致变红，直接点击该卡片上的 **“重新下载”** 即可原地复活！

**4. 结束使用**

当您今天的抓取任务完成，想彻底切断后台双引擎并释放端口资源时，只需一键化清理：
- **Mac / Linux**: 执行 `./stop.sh`
- **Windows**: 双击运行 `stop.bat`

## 📁 核心目录结构

本项目严格遵循高内聚低耦合的架构规范进行了深度瘦身重构。

- `/start.sh` & `/stop.sh`: 项目的命脉，跨语言环境一键启停脚本。
- `/RULES.md`: 本项目遵循的 AI 全局协作与开发编码规范。
- `/src/modules/web-ui/`: 基于 `React` + `Vite` 构建的现代化前端纯视觉交互层。
- `/src/modules/media-crawler/`: 基于 `Python` + `FastAPI` 构建的核心下载调度 API，内部承载了强悍的无头浏览器实例与各平台复杂的 JS 签名解析算法。
- `/docs/`: 提供给二次开发者的架构模块文档。

## 💡 使用小贴士 (Tips)

关于防封与 Cookie 保护：
当您在新版 Web 界面中点击**「拉起扫码登录」**时，后台会在您的本地真实弹出一个纯净的无痕浏览器供您用手机扫码。
扫码成功获取身份后，您可以尽情地批量下载视频。使用完毕后，只需点击**「关闭后台浏览器」**按钮即可悄声无息地释放内存。系统会在本地妥善为您加密缓存这一次的成功会话（Cookie），方便几天内免扫码复用！
